# JSX Practice Tasks

## Task 1: Display a Variable

Create a variable called `name` and display it inside an `<h1>` element.

---

## Task 2: Perform a Calculation

Display the result of `25 + 75` inside a `<p>` tag.

---

## Task 3: Call a Function

Create a function that returns `"Welcome to React"` and display it inside JSX.

---

## Task 4: Use a Ternary Operator

Create a variable `isAdmin`.

* If `true`, show `"Admin Panel"`.
* If `false`, show `"Access Denied"`.

---

## Task 5: Render an Array

Create an array of three favorite programming languages and display them using `map()`.

---

## Task 6: Practice the Single Root Rule

Create a component that displays a heading and a paragraph:

* First using `<div>`.
* Then replace the `<div>` with a React Fragment.

---

## Task 7: Convert `React.createElement()` to JSX

Convert the following into JSX:

```javascript
React.createElement(
  "h1",
  null,
  "Learning JSX"
);
```
